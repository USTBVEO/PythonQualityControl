# PythonQualityControl
准备开发一个用来实现新老7QC质量管理方法的python工具库，今日更新：直方图和正态拟合

做《现代质量工程学》作业的时候发现没有什么python好用的质量工具库，于是有了这个想法。

日期：2022年11月  
完成人：北航QEL实验室 杨威  

测试对象：北京35个空气监测站的污染物数据
